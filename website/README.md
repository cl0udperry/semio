# Sem.io Website

Professional website for Sem.io - AI Security Analysis Agent for CI/CD pipelines.

## 🚀 Features

- **Modern Design**: Clean, professional layout with gradient backgrounds
- **Responsive**: Works perfectly on desktop, tablet, and mobile
- **Interactive Demo**: Embedded Hugging Face Space for live demonstrations
- **Contact Forms**: Integrated contact forms with email functionality
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Fast Loading**: Optimized CSS and JavaScript for performance

## 📁 File Structure

```
website/
├── index.html          # Main homepage
├── styles.css          # All styling and responsive design
├── script.js           # Interactive functionality
└── README.md           # This file
```

## 🔧 Customization

### Colors
The website uses a consistent color scheme:
- Primary: `#667eea` (Blue gradient)
- Secondary: `#764ba2` (Purple gradient)
- Text: `#1a202c` (Dark gray)
- Background: `#f8fafc` (Light gray)

### Fonts
- Primary: Inter (Google Fonts)
- Fallback: System fonts

### Responsive Breakpoints
- Desktop: > 768px
- Tablet: 768px - 1200px
- Mobile: < 768px

## 🚀 Deployment Options

### Option 1: Static Hosting (Recommended)
